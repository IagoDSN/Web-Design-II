# 🖨️ 3D Print — Projeto de Web Design

## 📌 Sobre o projeto
O **3D Print** é um site desenvolvido com foco em serviços de impressão 3D, utilizando **Bootstrap** para garantir um design moderno, responsivo e prático.

---

## 🎯 Objetivos
- Divulgar serviços de impressão 3D  
- Mostrar projetos e modelos criados  
- Facilitar o contato com clientes  
- Criar um site bonito e fácil de usar  

---

## 🛠️ Tecnologias utilizadas
- HTML5  
- CSS3  
- JavaScript  
- Bootstrap 5  

---

## 📱 Funcionalidades
- Layout responsivo (mobile e desktop)  
- Navbar com menu de navegação  
- Seção de serviços  
- Galeria de modelos 3D  
- Formulário de contato  
- Rodapé informativo  

---

## 🎨 Design
- Uso de componentes do Bootstrap  
- Cores modernas (escuro + tons tecnológicos)  
- Layout limpo e organizado  
- Cards para exibir produtos  

---
